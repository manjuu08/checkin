## 🚀 インストール方法

スクリプトを利用するには、事前にお使いのブラウザへ [Tampermonkey](https://www.tampermonkey.net/) 拡張機能をインストールしてください。その後、上記の「**スクリプト一覧**」の表からお好みの方法でインストールできます。

### 方法1：GreasyFork 経由でインストール（推奨）
アップデートの管理が簡単になるため、こちらの方法を推奨しています。

1. 表の **GreasyFork** 列にある「インストール」リンクをクリックします。
2. 対象スクリプトの GreasyFork ページが開くので、緑色の「**スクリプトをインストール**」ボタンをクリックしてください。

### 方法2：GitHub から直接インストール
最新のソースコードを直接インストールしたい場合は、こちらをご利用ください。

1. 表の **GitHub (Raw)** 列にある「インストール」リンクをクリックします。
2. Tampermonkey のインストール確認画面が自動的に開きますので、「インストール」ボタンをクリックしてください。

> 💡 **Tip:** 私の公開しているすべてのスクリプトは、[GreasyFork のユーザーページ](https://greasyfork.org/ja/users/1618890-manjuu08) からも一覧で確認・インストールが可能です。
