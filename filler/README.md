LMS 一括解答入力ツール (Blackboard Bulk Answer Filler)

帝京大学LMSでのテスト・小テストにおける解答入力および管理を効率化するための Tampermonkey スクリプトです。

🔗 インストール (Installation)

  [Greasy Fork からインストールする](https://greasyfork.org/ja/scripts/585490)

機能概要 (Features)

    解答の一括入力: 「正/誤」や「A/B/C/D」などの解答リストを一括貼り付けし、自動的にチェックを反映します。

    解答の一括クリア: 選択済みの全解答をワンクリックで解除・リセットします。

    問題と解答の抽出・コピー: 採点後のレビューページから、問題文と解答をまとめて抽出し、クリップボードへコピーします。

使用方法 (How to use)
1. 解答の一括入力

    ページを開いた状態で、Tampermonkey アイコンから「正誤/選択問題の一括入力」を選択します。

    ポップアップ画面に、1行につき1つの解答（例: 正, A, B...）を貼り付けます。

    「記入を確定」ボタンを押すと、自動的に問題へのチェックが行われます。

2. 解答のクリア

    メニューから「選択済みの解答を一括クリア」を選択すると、ページ内のすべての選択状態が即座に解除されます。

3. 問題と解答のコピー

    メニューから「問題と解答を抽出してコピー」を選択すると、ページ内の全情報をクリップボードにコピーできます。バックアップや復習に便利です。

注意事項 (Important)

    本スクリプトは業務（学業）効率化を目的として作成されています。利用先のサイト規約および大学の学内規定を遵守の上、自己責任でご使用ください。

    Blackboard 側の仕様変更やUIアップデートにより、予告なく動作しなくなる場合があります。

    大量入力をする際は、問題の並び順とテキストの内容が一致していることを必ず確認してから実行してください。
