衛生報告入力自動化ツール (Hygiene Report Automation Tool)

衛生報告フォームの入力業務を効率化するための Tampermonkey スクリプトです。

🔗 インストール (Installation)

[Greasy Fork からインストールする](https://greasyfork.org/ja/scripts/585187)
1. 機能概要 (Features)

    衛生チェックの自動化: チェックボックスを自動的に全選択。

    体温の自動入力: 36.0℃〜36.5℃の範囲でランダムな値を自動設定。

    出勤者ID管理: 保存した出勤者IDを自動呼び出し。

    確認者ID支援: 確認者ID（033始まり）を自動スキャンし、リストからワンクリックで選択可能。

2. 使用方法 (How to use)

    出勤者IDの設定:

        ブラウザの Tampermonkey アイコンをクリックし、メニューから「⚙️ 出勤者IDを設定」を選択して自分のIDを保存してください。

    自動入力の実行:

        衛生報告ページを開き、「✅ 一括全自動入力」ボタンをクリックしてください。

    確認者の選択:

        確認者選択画面が表示されたら、リストから該当する確認者を選択してください。
